# CodeAlpha IoT Task 2

## Project Title
Motion Detection using PIR Sensor and Arduino

## Objective
To detect motion using a PIR sensor and turn an LED ON when motion is detected.

## Components Used
- Arduino Uno
- PIR Motion Sensor
- LED
- 220Ω Resistor
- Breadboard

## Working
The PIR sensor detects motion and sends a HIGH signal to the Arduino. The Arduino turns the LED ON. When no motion is detected, the LED turns OFF.

## Applications
- Smart Home Automation
- Security Systems
- Automatic Lighting
- Motion Detection
